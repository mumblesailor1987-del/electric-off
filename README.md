# TunisianH — Real-Time Electricity Outage Map for Tunisia

> **Tagline:** *Real-time electricity outage map for Tunisia, by citizens, for citizens.*

## Overview
**TunisianH** is a trilingual (Arabic / English / French), mobile-first, citizen-powered live outage map for Tunisia. It aggregates community signal reports, allows direct citizen claims, and provides live status maps, replay modes, historical statistics, and short-term estimates.
